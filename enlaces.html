Enlaces

La Web se basa en dos conceptos: el hipertexto y la hipermedia.

El hipertexto permite crear, agregar, enlazar y compartir información con una estructura no secuencial que proviene de diversas fuentes por medio de enlaces. Los enlaces también se conocen como hipervínculos, vínculos o ligas (en Sudamérica). Y en inglés se emplean los términos hyperlink y link.

Los nodos son las partes del hipertexto que contienen información accesible para el usuario. Por otro lado, los enlaces son las uniones o vínculos que se establecen entre nodos y facilitan la lectura secuencial o no secuencial por los nodos del documento. Por último, los anclajes son los puntos de activación de los enlaces.

Tipos de enlaces

  • Enlaces intradocumentales: es un enlace a un punto en el mismo documento en el que se define el enlace.
  • Enlaces extradocumentales: a un punto en otro documento.

Uniform Resource Locator

El URL es el acrónimo de Uniform Resource Locator, Localizador uniforme de recurso, y es la forma estandarizada de hacer referencia a un documento, a un recurso, en la Web.

Una URL se compone de:

  • El servicio o protocolo que se está empleando.
  • De las credenciales, nombre de usuario y contraseña, con las que se realiza la conexión al servicio.
  • El host o servidor al que se accede.
  • El puerto al que se realiza la conexión.
  • La ruta de acceso al recurso.
  • Finalmente, el nombre del recurso, que suele ser una página web, pero que podría ser cualquier tipo de documento.

Etiquetas

Atributos:

  • href="": Aquí se indica la dirección a la pagina que se quiere enviar, ademas si se quiere ubicar a una parte del sitio, usar # y el id del sitio al que se quiere enviar.(Ver codigo fuente de la página principal)
  • target: Para indicar si se abre en la misma pagina o en otra pestaña, para esta última opción se usa"_blank" y la primera opción esta como default, es decir, no hay necesidad de poner el atributo target.
