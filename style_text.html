Estilo de Texto

font-family

Tipo de letra
 

font-size

Tamaño de letra

font-weight

Grosor de la letra
 

font-style

Estilo de la letra
 

font-variant

La propiedad font-variant selecciona entre los aspectos normal y small-caps para la familia de fuente determinado.
 

text-align

Alinear el texto
 

text-indent

Sangría a la linea del texto
 

text-decoration

Para el subrayado o decoracion del texto
 

text-transform

Para poner el texto en mayuscula, minuscula o capitalizarlo (Primera letra en mayuscula)
 

letter-spacing

Espacio entre letras

word-spacing

Espacio entre palabras

line-height

Interlineado

white-space

Para que el texto se muestre literal como se escribe en el código HTML
