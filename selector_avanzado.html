Descendientes Directos (>)


Si entre el selector de la etiqueta superior, en este caso la lista con identificador “lista” y el de las etiquetas anidadas incluimos el carácter “mayor que” (>), solo se seleccionarán los elementos li que sean descendientes directos.

Desafortunadamente, ni si quiera en CSS3 existe el carácter “menor que” (<) para seleccionar etiquetas padre.

Elementos contiguos


En el ejemplo puedes ver cómo funciona el selector + entre dos elementos. En este caso, se aplicarán los estilos a todos los elementos span que estén tras una imagen con clase “new”. Ojo, solo a los inmediatamente contiguos.

:first-child


El selector :first-child de forma conjunta a un elemento, en el ejemplo li, aplica la regla al primer elemento de ese tipo, que además sea el primero anidado, en este caso en la etiqueta ul.

:first-letter


El selector :first-letter es bastante más intuitivo, se aplica a la primera letra del texto del elemento seleccionado. Puede servir para simular letras capitales.

:first-line


El selector :first-line se aplica a la primera línea del texto del elemento seleccionado.

Selector por Atributo

 
Si especificamos entre corchetes el nombre de un atributo, en este caso “title”, los estilos se aplicarán a todos los elementos que tengan un atributo “title”, no importa cuál sea su valor (Ver imagen izquierda). Si quieres especificar más, puedes igualar el atributo a una cadena de texto, en este caso escribimos “title=Buscador” (Ver imagen derecha).

Atributos que empiece por esa "palabra"


Para seleccionar elementos cuyo atributo empiecen por una palabra, como puede ser “top” en el caso de los identificadores de los enlaces del ejemplo, añadimos la barra vertical(|).

Atributos que contengan esa "palabra"


Mediante la virguilla o tilde de la eñe(~), podemos especificar que el atributo contenga una palabra, esta vez el valor del elemento sí puede estar separado por espacios.

:lang


:lang y entre paréntesis el código del idioma según el estándar ISO 639, selecciona todas las etiquetas cuyo valor del atributo “lang” coincida con la especificada entre paréntesis, como en el ejemplo.

Selectores CSS3

:last-child: Selecciona el último elemento de la lista.
:nth-child(n): Seleccionan el hijo enésimo empezando desde el principio.
:nth-last-child(n): Seleccionan el hijo enésimo empezando desde el final.
:first-of-type: Selecciona al primer hijo (tipo).
:last-of-type: Selecciona al último hijo (tipo).
:nth-of-type(n): Seleccionan el hijo enésimo empezando desde el principio (tipo).
:nth-last-of-type(n): Seleccionan el hijo enésimo empezando desde el final(tipo).
~: Aplica a todos los hermanos de ese tipo.
